# Прогнозирование цен на автомобили

## Описание проекта
Этот проект направлен на **прогнозирование цен на автомобили** с использованием алгоритмов машинного обучения. Модель предсказывает цену автомобиля на основе таких характеристик, как пробег, год выпуска, тип кузова, количество цилиндров, тип коробки передач и других факторов.

В ходе проекта были использованы различные методы предобработки данных, включая обработку пропусков, выбросов, создание новых признаков и кодирование категориальных переменных.

Модели машинного обучения, такие как **Линейная регрессия**, **Случайный лес** и **Градиентный бустинг**, были обучены и протестированы для предсказания цен автомобилей.
